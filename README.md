# Python-slot-machine
A Python-based emoji slot machine game with betting system, random spins, and win calculation logic.

Desc: # Emoji Slot Machine (Python)

A simple terminal-based slot machine game built using Python.
This project simulates a casino-style slot machine where users can deposit money, place bets, and spin to win.

## Features

* Deposit system 
* Betting on multiple lines
* Random slot spin logic 
* Emoji-based slot display 
* Win calculation system
* Continuous gameplay until user quits

## Tech Used

* Python
* Random module

## How to Play

* Enter deposit amount
* Choose number of lines (1–3)
* Place your bet
* Spin the slot machine
* Win if symbols match across a line!


## Notes

This is a beginner-friendly project created to practice Python fundamentals such as:

* Functions
* Loops
* Conditionals
* Data structures
